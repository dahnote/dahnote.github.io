理財小白必懂的股市常識名詞懶人包

發表於 2021-12-03 更新於 2022-02-28 分類於 股市基礎 閱讀次數：

股市中漲與跌，是獲利或虧損的基礎，但很多時候你一旦只知道漲與跌這兩個名詞，最後恐怕就是準備被割的韭菜。在這篇懶人包中會把盤勢、交易、K線圖、特殊事件，簡單的找出常識名詞給大家當懶人包，本文未來持續更新，如果你有疑問歡迎來信做詢問。

盤勢相關

  1. 上漲:今日收盤價格，大於昨日收盤價格
  2. 下跌:今日收盤價格，小於昨日收盤價格
  3. 平盤:今日收盤價格，等於昨日收盤價格
  4. 牛市:上漲趨勢，也有人稱為多頭格局
  5. 熊市:下跌趨勢，也有人稱為空頭格局

K線圖怎麼看?

  1. K線圖的組成: 開盤價格、盤中最高價、盤中最低價、收盤價格
  2. 紅K:收盤價格大於開盤價格；股價上漲，用紅色來繪製
  3. 黑K:收盤價格小於開盤價格；股價下跌，用綠色來繪製
  4. 十字線:收盤價格等於開盤價格，表示買方與賣方對於價格的影響力不分上下。
  5. K線週期:除了單日的K線，多數看盤軟體也能夠切換分K線、週K線、月K線

交易相關

  1. 搓合:由證券交易所系統進行媒合，讓有出價的買方與賣方，在雙方可接受的價格進行成交。
  2. 融資:借錢買進股票，借錢對向為券商，利息6%上下，每1萬本金可借1.5萬元，到期時需要本與息同時償還。
  3. 融券:看空公司的交易方式， 先向券商借出股票，當高股票高價時賣出，低價時買入，到期時償還當初借出數量的股票和利息。
  4. 軋空:當放空時股票不跌反漲，放空方被迫虧損買入股票償還。

特殊事件

  1. 股東會:公司投資人的會議，散戶可於此會議詢問經營階層，也可能會遇到董監選舉，延伸的事件會有委託書徵選。當然還有大家會關注的紀念品也可以拿股東會通知書去兌換。
  2. 法說會:簡單來說是業績的說明會，會有公司未來的發展規劃或是財務說明，產業的願景說明。目前規定上市公司每年都需要開一次法說會。
  3. 封關日:台灣最長的國定假日為農曆年假，因應股市的T+2日機制，會在農曆年前休市停止交易，因此最後一天的交易日通常都稱為封關日。
  4. 紅盤日:農曆年假的結束的第一天交易日，因為取其吉祥的象徵，又稱為紅盤日。
